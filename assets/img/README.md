# Dossier Images

Ce dossier contient les images pour illustrer le portfolio.

## Instructions pour ajouter vos images

1. **Pour les compétences** : Placez les images dans ce dossier avec les noms :
   - `bdd-schema.svg` ou `bdd-schema.png` - Schéma de base de données
   - `reseau-gephi.png` - Graphique de réseau Gephi
   - `lexicometrie-afc.png` - Analyse factorielle des correspondances
   - `nlp-tfidf.png` - Visualisation TF-IDF
   - `visualisation.png` - Exemple de visualisation
   - `interdisciplinarite.png` - Schéma interdisciplinaire

2. **Pour les projets** : Placez les images avec les noms :
   - `mshs-t-reseau.png` - Réseau MSHS-T
   - `histoire-art-reseau.png` - Réseau peintres itinérants
   - `tohoku-analyse.png` - Analyse Tohoku

   - `noblesse-reseau.png` - Réseau noblesse
   - `japon-isolement.png` - Analyse Japon
